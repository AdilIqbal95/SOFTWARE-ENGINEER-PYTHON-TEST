#Description:
A simple REST endpoint that returns the sum of the list: list(range(10000001)).

Key files:
main.py
test_1.py

These files are to be run simultaneously. First run main.py. This will start the server.
test_1.py is a test file which will send a post request to the specified endpoint, obtain
and return the sum of the given list.

##Packages used:
Flask       1.1.2
json5       0.8.4
requests    2.22.0
